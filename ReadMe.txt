

Quoridor 2017

*************

Pentru a putea rula jocul Quoridor trebuie sa descarcati toate cele 3 fisiere header (aflate in repository):
 - "Data Structures and Constants.h"
 - "Functions.h"
 - "Input and Output.h"

De asemenea, este necesara descarcarea unui pachet SFML adecvat versiunii de Visual Studio:
 - http://www.sfml-dev.org/download/sfml/2.4.1/

Apoi este necesara legarea fisierelor in proiectul aplicatiei. Se poate urmari urmatorul tutorial:
 - https://www.youtube.com/watch?v=eTe7IVncH5E

In proiectul aplicatiei se vor plasa si fisierele dll necesare, care se gasesc in folderul SFML descarcat (directorul bin),
fontul "arial.ttf" si imaginile "q.jpg", "q1.jpg", "q2.jpg", "q3.jpg" ce se gasesc in repository-ul de pe github.



Pentru a primi informatii despre regulile si desfasurarea jocului se poate descarca fisierul Rules.docx aflat in repository.
